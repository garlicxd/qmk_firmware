## Characters

#### Basic Symbols

> [!NOTE]- All basic symbols in order of importance in Rust
> / _ : ( ) , . { } = ; " > \` ' - & < ! [ ] # | * \ ~ $ ? + ^ @ %

> [!NOTE] Things that are important to type with numbers.
> % ^ ( ) [ ] { } < > = + - * / : ~ 
> Exclusive: % ^ ~

> [!NOTE] Things that are important to program.
> ( ) [ ] { } < > & | = + - * / : ! ? " \
> Exclusive: & |

> [!NOTE] Important to both
> < > = + - * / :

> [!NOTE] Important to neither
> @ # $ \`
